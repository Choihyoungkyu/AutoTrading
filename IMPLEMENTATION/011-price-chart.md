## 새 API 엔드포인트

```
GET /api/stock/kr/{code}/price-history?period={1d|1w|1m|1y}
```

### 기간 매핑

| 버튼 | period | 조회 범위 |
|---|---|---|
| 일 | `1d` | 최근 30 캘린더일 (약 20 거래일) |
| 주 | `1w` | 최근 91일 (약 13주) |
| 달 | `1m` | 최근 365일 ← 기본값 |
| 년 | `1y` | 최근 1095일 (3년) |

### 응답 예시

```json
{
  "code": "005930",
  "period": "1m",
  "count": 245,
  "data": [
    { "date": "2025-06-24", "open": 68000, "high": 70000, "low": 67500, "close": 69500, "volume": 12000000, "change": 0.72 },
    ...
  ]
}
```

## 프론트엔드 변경

### 추가된 UI 요소

- **주가 차트 카드**: 재무 분석 섹션 위에 삽입
- **기간 버튼**: 일 / 주 / 달 / 년 (활성 버튼은 다크 배경으로 강조)
- **Chart.js 캔버스**: 높이 320px 라인 차트

### Chart.js 설정 포인트

| 항목 | 설정 |
|---|---|
| 라이브러리 | Chart.js 4.4.0 (CDN) |
| 차트 타입 | `line` + `fill: true` (그라디언트 배경) |
| 포인트 | 데이터 60개 초과 시 `pointRadius: 0` (성능) |
| 툴팁 | 날짜 + `₩` 종가 포맷 |
| x축 | 최대 8개 눈금 (레이블 겹침 방지) |
| y축 | `₩` 천단위 콤마 포맷 |

### JS 함수

```javascript
loadPriceChart(period)  // 버튼 클릭 시 호출, 기존 차트 destroy 후 재생성
```

페이지 로드 시 `loadPriceChart('1m')` 자동 실행.
