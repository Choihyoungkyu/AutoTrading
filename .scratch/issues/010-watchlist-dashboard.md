# 관심 종목 대시보드 (우수 기능)

## What to build

사용자가 관심 있는 종목들을 저장하고, 한 화면에서 여러 종목의 분석 결과를 한눈에 비교할 수 있는 대시보드를 만든다. 직장인이 "오늘 봐야 할 5개 종목의 추천 신호가 뭐지?"를 빠르게 파악할 수 있게 한다.

**End-to-end 동작:**
```
대시보드 → "관심 종목" 탭
→ 저장된 종목 목록 (삼성전자, NAVER, SK Hynix 등)
→ 종목별 카드 표시:
  [005930] 삼성전자  | BUY | 현가 70000 | 목표 80500 | PER 8.5
  [035420] 네이버    | HOLD | 현가 250000 | 목표 275000 | PER 15.2
  [000660] SK Hynix | SELL | 현가 120000 | 목표 100000 | PER 6.1

또는 테이블 형식:
종목     | 신호  | 현가    | 목표가  | PER | 뉴스심리
---------|-------|---------|---------|-----|--------
삼성전자  | BUY  | 70000  | 80500  | 8.5 | 긍정
네이버    | HOLD | 250000 | 275000 | 15.2| 중립
SK Hynix | SELL | 120000 | 100000 | 6.1 | 부정
```

## Acceptance criteria

- [ ] 관심 종목 목록 저장 (로컬스토리지 또는 백엔드 DB)
  - 추가 버튼: 종목 검색 후 추가
  - 제거 버튼: 목록에서 제거
- [ ] 관심 종목들의 분석 일괄 조회 (병렬 처리)
- [ ] 테이블 또는 카드 형식으로 표시
  - 종목코드 | 종목명 | 현가 | 목표가 | 추천신호(BUY/HOLD/SELL) | PER | 뉴스심리
- [ ] 신호별 색상 구분 (BUY: 초록, HOLD: 황색, SELL: 빨강)
- [ ] 정렬 기능 (신호별, PER순 등)
- [ ] "한눈에 보기" 텍스트 (예: "관심 5개 중 BUY 2개, HOLD 1개, SELL 2개")
- [ ] 선택한 종목 클릭 → 상세 분석 페이지 이동
- [ ] 새로고침 버튼 (최신 데이터 조회)
- [ ] 모바일 반응형 (선택사항)
