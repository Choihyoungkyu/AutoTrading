# 웹 대시보드 UI (삼성전자 리포트)

## What to build

Vue.js로 웹 대시보드를 만든다. 종목 입력 폼에서 삼성전자 코드(005930)를 입력하면 한 화면에 모든 분석 결과가 시각화되어 나타난다. 직장인이 점심시간 5분 만에 투자 결정을 내릴 수 있게 한다.

**End-to-end 동작:**
- 웹 페이지 로드 (http://localhost:3000)
- 종목 입력 폼에 "005930" 입력 → Enter
- 2~3초 후 한 화면에 다음이 모두 표시:
  - 종목명 & 현재가
  - 재무 지표 (테이블: PER, PBR, ROE, EPS, 부채비율, 배당수익률)
  - 차트 시각화 (이동평균선 그래프)
  - 기술적 신호 (Buy/Sell/Hold)
  - 뉴스 요약 (최근 3개 헤드라인)
  - 종합 추천 (크고 눈에 띄는 "BUY" 배지)
  - 목표가·손절 (녹색 목표가, 빨간색 손절가)

## Acceptance criteria

- [ ] Vue 프로젝트 초기화 (npm 또는 yarn)
- [ ] 종목 입력 폼 UI (input + 검색 버튼)
- [ ] 로딩 상태 표시 (분석 중 로더)
- [ ] 재무 지표 섹션 (테이블 형식)
- [ ] 기술적 분석 섹션
  - 이동평균선 차트 (Chart.js 또는 유사)
  - RSI, MACD 표시 (또는 기술적 신호 텍스트)
- [ ] 뉴스 섹션 (최근 3개 헤드라인)
- [ ] 종합 추천 섹션 (Buy/Hold/Sell 배지 + 점수)
- [ ] 목표가·손절 섹션 (명확한 숫자 표시)
- [ ] 기본 스타일링 (어두운 테마 또는 밝은 테마)
- [ ] 오류 처리 (종목 없음, API 연결 실패 메시지)
- [ ] 반응형 디자인 (데스크톱 기준)
