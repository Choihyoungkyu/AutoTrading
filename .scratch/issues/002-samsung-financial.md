# 삼성전자 재무 분석 엔드포인트

## What to build

PYKRX를 통해 삼성전자(005930)의 재무 지표를 수집하고, 동종업계(반도체) 평균과 비교하여 저평가/고평가를 판단하는 엔드포인트를 만든다. 직장인이 "이 기업은 현재 저평가인가?"라는 질문에 답할 수 있게 한다.

**End-to-end 동작:**
```
GET /analyze/005930/financial
→ 
{
  "per": 8.5,
  "pbr": 1.2,
  "roe": 12.5,
  "eps": 3500,
  "debt_ratio": 45.2,
  "dividend_yield": 2.1,
  "industry_avg": { "per": 15.0, "pbr": 1.8, ... },
  "verdict": "저평가"
}
```

## Acceptance criteria

- [ ] PYKRX에서 삼성전자 재무 데이터 수집 (최근 분기 기준)
- [ ] PER, PBR, ROE, EPS, 부채비율, 배당수익률 계산
- [ ] 동종업계(반도체) 평균 데이터 수집 및 비교
- [ ] `/analyze/005930/financial` 엔드포인트 구현
- [ ] 저평가/고평가 판정 로직 구현
- [ ] 고정 재무 데이터로 단위 테스트 작성 (mock)

## Blocked by

- 001: 프로젝트 초기화 & 데이터 수집 기본 구조

## User Stories

- 1: 종목코드 입력 시 재무 지표를 자동으로 계산 및 표시
- 2: 현재가 대비 PER, PBR이 저평가/고평가인지 한눈에 보기
- 3: 동종업계 평균 지표와 비교된 결과를 보기
