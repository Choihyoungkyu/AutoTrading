# 목표가 & 손절 라인 자동 제안

## What to build

현재 주가를 기반으로 합리적인 목표가와 손절 라인을 자동 계산한다. 직장인이 "얼마에 사서 얼마에 팔지"를 사전에 명확하게 할 수 있게 한다.

**End-to-end 동작:**
```
GET /analyze/005930/price-target
{
  "current_price": 70000,
  "expected_return": 0.15,    // 기대수익률 15%
  "max_loss": 0.10            // 최대손실률 10%
}
→
{
  "target_price": 80500,      // 70000 × (1 + 0.15)
  "stop_loss": 63000,         // 70000 × (1 - 0.10)
  "risk_reward_ratio": 1.5    // (80500 - 70000) / (70000 - 63000)
}
```

## Acceptance criteria

- [ ] 현재 주가 입력받기
- [ ] 기대수익률 설정 (기본값: 15%)
- [ ] 최대손실률 설정 (기본값: 10%)
- [ ] 목표가 계산: 현재가 × (1 + 기대수익률)
- [ ] 손절가 계산: 현재가 × (1 - 최대손실률)
- [ ] 리스크/리워드 비율 계산
- [ ] `/analyze/005930/price-target` 엔드포인트 구현
- [ ] 고정 가격 데이터로 단위 테스트 작성 (계산 검증)
