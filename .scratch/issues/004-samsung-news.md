# 삼성전자 뉴스 분석

## What to build

최신 뉴스를 자동으로 수집하고, 호재/악재 키워드를 분석하여 시장 심리를 점수화한다. 뉴스 크롤링이 실패해도 캐시 데이터로 폴백하여 시스템이 작동하도록 한다.

**End-to-end 동작:**
```
GET /analyze/005930/news
→
{
  "score": 0.65,  // -1.0(악재) ~ 1.0(호재)
  "sentiment": "positive",
  "headlines": [
    { "title": "삼성전자, 3분기 영업이익 사상 최대", "score": 0.9 },
    { "title": "반도체 경기 전망 불확실", "score": -0.3 }
  ],
  "source": "live"  // or "cache" if yfinance API fails
}
```

## Acceptance criteria

- [ ] yfinance 뉴스 API로 삼성전자 최신 뉴스 수집 (5개 이상)
- [ ] 국내 크롤링 (선택사항): 네이버 금융 뉴스 추가 수집
- [ ] 호재/악재 키워드 사전 구축 (규칙 기반)
  - 호재: "상승", "사상 최대", "호실적", "신제품" 등
  - 악재: "하락", "적자", "부진", "차질" 등
- [ ] 각 헤드라인 감정 점수 계산 (-1.0 ~ 1.0)
- [ ] 전체 점수 합산 (평균)
- [ ] `/analyze/005930/news` 엔드포인트 구현
- [ ] 크롤링 실패 시 캐시 데이터 폴백 로직
- [ ] 고정 헤드라인으로 단위 테스트 작성 (mock)
