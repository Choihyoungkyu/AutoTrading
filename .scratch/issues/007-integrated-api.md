# 통합 분석 리포트 API (Primary Seam)

## What to build

위의 모든 분석 모듈(재무, 차트, 뉴스, 추천, 목표가)을 한 번의 API 호출로 통합한다. 직장인이 종목 코드 하나만 입력하면 필요한 모든 정보를 5초 이내에 얻을 수 있다.

**End-to-end 동작:**
```
GET /analyze/005930
→
{
  "stock_code": "005930",
  "stock_name": "삼성전자",
  "current_price": 70000,
  "financial": { ... },
  "chart": { ... },
  "news": { ... },
  "recommendation": { ... },
  "price_target": { ... },
  "timestamp": "2026-06-24T15:30:00Z"
}
```

## Acceptance criteria

- [ ] 종목 코드 유효성 검증 (존재하는 종목인가)
- [ ] 002~006의 모든 분석을 병렬로 실행
- [ ] 결과를 하나의 JSON으로 통합
- [ ] 응답 시간 5초 이내 (캐싱 활용)
- [ ] 오류 처리
  - 존재하지 않는 종목: 404
  - 데이터 수집 실패 (일부): 부분 캐시 데이터 반환
  - 전체 데이터 수집 실패: 500 + 오류 메시지
- [ ] GET `/analyze/<stock_code>` 엔드포인트 구현
- [ ] 엔드투엔드 테스트 (정상 종목, 잘못된 종목, 부분 실패)

## Blocked by
