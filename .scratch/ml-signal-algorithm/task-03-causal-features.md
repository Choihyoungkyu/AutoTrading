# T03: 인과적 피처 빌더 (S2)

## What to build

시점 T까지의 봉만으로 **인과적으로** 기술지표 피처를 계산한다. 미래 봉이 절대 새지 않음을 코드로 보장한다.

**End-to-end 동작:**
```
build_features(OHLCV_up_to_T) -> 피처 벡터 (이동평균/RSI/거래량 파생/변동성 등)
```

## Acceptance criteria

- [ ] 이동평균·RSI·거래량 파생·변동성 등 기술지표를 과거 봉만으로 순차 계산
- [ ] 전체 기간 기준 정규화 금지 (그 시점에 미래 데이터 없음)
- [ ] 수정주가 사용 시 미래 액면분할/배당 소급 정보 유입 주의
- [ ] 지표 워밍업 구간의 결측(NaN) 명시적 처리
- [ ] 피처 목록 확정 및 각 피처의 "T 이후 불변" 계약 문서화

## Seam (테스트 필수)

미래참조 부재 검증:
- 시점 T 피처 계산 → 이후 봉(T+1..) 추가 후 재계산 → **T 시점 피처 동일**해야 함
- 워밍업 구간 NaN 처리

## Blocked by

없음 (합성 데이터로 독립 구현·테스트 가능)
