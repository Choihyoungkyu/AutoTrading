# T02: 라벨 생성기 (S1, 실현손익 시뮬레이터)

## What to build

시점 T에 매수했다고 가정하고 실제 매매 규칙을 시뮬레이션해 **"비용 제외 실현손익이 (+)였나"**를 0/1 라벨로 만든다. **라벨 = 실제로 번 돈.** 이 프로젝트의 성패가 갈리는 심장부.

**End-to-end 동작:**
```
make_labels(OHLCV, stop_loss=-0.03, horizon=5, cost=0.007) -> Series[0/1]
```

## Acceptance criteria

- [ ] 진입: T 종가 매수
- [ ] 청산: T+1~T+5 중 장중 저가가 -3% 손절선 터치 시 그 시점 손절가 청산, 아니면 T+5 종가 청산
- [ ] 라벨: `실현손익(%) - 왕복거래비용(%) > 0` → 1, 아니면 0
- [ ] 비용(수수료+거래세+슬리피지)은 파라미터(기본 0.5~1%)
- [ ] "T+5 종가만 보는" 방식(look-ahead A) 아님 — 경로 의존적 실현손익
- [ ] 시계열 끝부분(미래 5일 없음) 라벨 생성 불가 처리

## Seam (테스트 필수)

합성 가격 시나리오로 전 경로 검증:
- 5일 내내 상승 → 손절 미발동, T+5 청산, 비용 초과 → 라벨 1
- 2일차 -3% 터치 후 반등, T+5 종가 +3% → 손절 청산 → **라벨 0** (라벨=돈 검증)
- 소폭 상승(+0.4%)이 비용 미달 → 라벨 0
- 손절선/비용선 경계값

## Blocked by
