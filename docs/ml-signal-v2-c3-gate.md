# C3: 실험 종합 + C→B/A 라우팅 게이트 판정

> v2 로드맵 Phase C 종료 관문. 이슈: `.scratch/ml-signal-algorithm-v2/C3-experiment-routing-gate.md`
> 설계: `docs/superpowers/specs/2026-07-08-ml-signal-v2-design.md` · 선행: `docs/ml-signal-v2-d3-gate.md`

## 무엇을 쟀나

D3 게이트(GO to C) 후, 큰 투자(B/A) 전에 "이 시장에서 뭐라도 이기나"를 싸게 확인하는
두 실험을 **동일 하네스**(`src/ml/pipeline.py`)·**동일 평가**(`reliability_report`)로
격리 측정했다. 각 실험은 D 베이스라인과 **변수 하나만** 다른 A/B다.

- **C1 (재무 min-viable):** D와 유니버스 동일(KOSPI 거래대금 상위 30, 연 1회 재선정),
  **재무 피처 4개 추가**(`fund_per`, `fund_pbr`, `fund_div`, `fund_earnings_yield`).
  공시 시차는 `join_fundamentals_asof`(merge_asof backward)로 엄수 — 미래 공시가
  과거 봉에 새지 않음(누수 회귀 테스트로 강제).
- **C2 (중소형주 min-viable):** D와 피처 동일(차트 16개), **유니버스만 교체**
  (KOSPI 거래대금 100~140위 밴드, min_value 5억원, 연 1회 재선정).

두 실험 모두 2019~2024, 5폴드, 락박스(2024) 봉인, HORIZON=5, COST=0.007.

이기지 못했다(초과수익 0/4, 폴드간 Sharpe 음수).

## 판정: **STOP**

설계의 라우팅 규칙: *C1 승 → A / C2 승 → B / 둘 다 승 → B 먼저 / **둘 다 실패 → STOP,
접근 전면 재검토**.* 두 실험 모두 실패했으므로 **STOP**.

- **B(유니버스 확대)·A(재무 풀셋·뉴스)로 진행하지 않는다.** 값싼 탐색에서 어느 방향도
  신호를 보이지 않았으므로, 무거운 투자를 정당화할 근거가 없다.
- B1·B2·A1·A2는 `needs-triage`로 **동결**(승격하지 않음).

### 왜 STOP이 견고한 결론인가

1. **세 방향 모두 무엇도 못 이김.** D(차트·대형주), C1(재무·대형주), C2(차트·중소형주)
   — 서로 다른 세 가설이 전부 기준선 미달. 우연이라기엔 일관된다.
2. **생존 편향이 아직 성능을 부풀리는 방향인데도 엣지가 없다.** 상폐·거래정지 종목은
   미포함(B2 미실행)이라 현재 수치는 **낙관 편향**된 상태다. 그 낙관적 조건에서조차
   엣지가 없으므로, 편향을 제거(B2)하면 결과는 더 나빠질 뿐 — B로 가도 반전 가능성 낮음.
3. **부호가 일관되게 음수**(양 실험 Sharpe −0.77/−0.82)라 소표본 노이즈가 아니다.

즉 "일봉 차트 + 기본 point-in-time 재무로, 유동성 있는 한국 주식을, 5일·-3%손절·현실
비용 규칙으로 매매"하는 이 문제 설정에는 **활용 가능한 엣지가 없다**는 것이 v2의 정직한
결론이다. 이는 v1의 결론을 세 방향으로 확장 검증해 재확인한 것이다.

## 다음 (인간 판단 필요 — ready-for-human)

STOP은 로드맵의 정상 종료다. 재검토는 "문제 설정 자체를 바꿀 것인가"의 결정이며,
값싼 피처 추가로 해결되지 않음이 드러났다. 재검토 시 고려할 방향(택1 이상, 별도 기획):

- **라벨/지평 변경**: 5일·-3% 규칙이 신호를 지운 것일 수 있음(다른 horizon·손절·목표).
- **대체 데이터**: 일봉 밖 — 체결/호가(order flow), 인트라데이, 대체데이터. 수집·누수
  통제 비용이 큼.
- **정직한 종료**: 이 데이터·규칙에는 엣지가 없음을 받아들이고 규칙 엔진 병존(ADR 0008)
  유지, ML 신호 미노출 지속.

이 판단은 도메인·비용 트레이드오프가 걸려 인간 결정 사항이다.
