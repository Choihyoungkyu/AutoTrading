# D3: 신뢰성 리포트 + D→C 게이트 판정

> v2 로드맵 Phase D의 종료 관문. 이슈: `.scratch/ml-signal-algorithm-v2/D3-reliability-report-gate.md`
> 설계: `docs/superpowers/specs/2026-07-08-ml-signal-v2-design.md`

## 무엇을 쟀나

D1(point-in-time 분할조정)·D2(시점별 유니버스 재선정·5폴드·락박스)로 강화한
하네스에서, v1의 "엣지 없음" 결론이 **통계적으로 믿을 만한지**를 판정한다.

- 유니버스: 연 1회 point-in-time 재선정(거래대금 상위 30, 데모 서브셋), 2019~2024
- 리밸런스 6회 → 합집합 93종목, 시점별 멤버십 필터 후 41,985행
- walk-forward 5폴드, **락박스(2024)는 봉인** — 본 판정에서 개봉하지 않음
- 평가 폴드: 4개(2020~2023)

## 신뢰성 리포트 (실측)

| 지표 | 값 | 해석 |
|---|---|---|
| 성공 판정 | **False** (0/4 승리, 통과기준 3) | 두 기준선을 마진(0.014)으로 못 이김 |
| 전략순수익 평균 | **-0.00265** | 음(-) |
| 전략순수익 표준편차 | 0.00449 | — |
| 편차 범위(min~max) | -0.0073 ~ +0.0035 | 폴드별 진폭 |
| **기준선 초과 폴드** | **0 / 4** | **한 번도** 더 나은 기준선을 못 이김 |
| 폴드간 Sharpe | **-0.59** | 음(-) |
| deflated Sharpe 통과 | False | 우연의 기대치조차 못 넘음 |

폴드별 기준선(다 사기·지수 중 큰 값) 대비 초과수익: `[-0.0029, -0.0036, -0.0024, -0.0067]`
— **4폴드 모두 음수**. 전략이 '다 사기'보다는 조금 낫지만, 지수 보유는 한 번도
이기지 못했다.

## 판정: **GO to C**

두 가지가 동시에 성립한다.

1. **측정은 안정적이다.** 초과수익 부호가 4폴드 내내 일관되게 음수다(노이즈가
   부호를 뒤집지 않음). 즉 "엣지 없음"은 소표본 우연이 아니라 **재현되는 결론**이다.
2. **엣지는 명백히 부재하다.** 평균·Sharpe 음수, 기준선 초과 0/4.

설계의 게이트 규칙(측정 안정 + 여전히 엣지 없음 → GO to C)에 정확히 해당한다.
차트 전용·대형주 신호로는 이 시장을 이길 수 없음이 **신뢰할 수 있게** 확인됐으므로,
**신호(재무)나 유니버스(중소형주)를 바꾸는 Phase C로 진행**한다.

- C1(재무 min-viable)·C2(중소형주 min-viable) 착수, 두 이슈를 활성화.
- C의 결과(C3 라우팅 게이트)에 따라 B 또는 A로 분기.

## 남는 한계 (다음 단계로)

- **n_trials=1(이번 실행 기준).** 앞으로 설정을 튜닝하며 시도가 늘면 deflated Sharpe로
  반드시 보정해 보고할 것(현재 임계치 로직 구현됨: `src/ml/reliability.py`).
- **락박스 미개봉.** 2024 구간은 봉인 유지. 최종 배포 판정(A3)에서 1회만 개봉한다.
- **생존 편향 잔존.** 상폐·거래정지 종목 미포함 → 성능이 낙관적. B2에서 완화.
- **데모 서브셋(top_n=30).** 코드는 100~200 지원. 결론의 방향은 확대해도 바뀌지
  않을 가능성이 높으나, C 단계에서 규모를 키우며 재확인한다.
